# 🚕 Uber Ride Data Analysis Dashboard

## 📌 Project Overview
This project analyzes Uber ride data to uncover insights related to revenue trends, booking patterns, and customer demand behavior. The dataset was cleaned and processed using Python, and an interactive dashboard was built using Tableau to visualize key metrics.

The goal of this project is to demonstrate an end-to-end data analysis workflow, from raw data to actionable insights.

---

## 🎯 Objectives
- Analyze ride data to identify revenue and demand patterns  
- Understand peak booking hours and ride behavior  
- Build an interactive dashboard for business insights  
- Demonstrate data cleaning, analysis, and visualization skills  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Tableau  
- Git & GitHub  

---

## 🧹 Data Cleaning & Preprocessing
Data preprocessing was performed using Python:
- Removed missing and inconsistent values  
- Cleaned and formatted date/time fields  
- Standardized numerical columns (fare, distance)  
- Prepared dataset for visualization in Tableau  

---

## 📊 Dashboard Features
The Tableau dashboard provides insights such as:

- Total Bookings and Revenue  
- Revenue trends over time (daily and monthly)  
- Hourly demand analysis (peak booking hours)  
- Top pickup locations by revenue and volume  
- Ride distance distribution  
- Fare vs Distance relationship  

---

## 🔍 Key Insights
- Identified peak demand hours, primarily during evening time slots  
- Discovered high-revenue pickup locations contributing significantly to total earnings  
- Observed a relationship between ride distance and fare pricing  
- Analyzed booking trends to understand customer demand patterns over time
